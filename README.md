# Control Room Attendance Portal

Employee Attendance and Workforce Management System for AVSOFT CORPORATION.

## Features

* GPS Based Attendance
* Face Verification
* Selfie Verification
* Shift Management
* Weekly Off Management
* Comp-Off Tracking
* Leave Management
* Salary Calculation
* Employee Self Registration
* HR Approval Workflow
* Notification Preferences

## Technology Stack

* Node.js
* Express.js
* TypeScript
* PostgreSQL
* Prisma ORM
* JWT Authentication
* Progressive Web App (PWA)
